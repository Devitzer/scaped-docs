# Help Command

This command **initializes** a plugin by creating the necessary files to get started with one.

## Usage

```bash title="Command Terminal"
scaped init (PluginName) (Directory) (--typescript)
```

## Arguements
### Required
#### PluginName
Should be __one__ word. If you need it to be two you can do something like "my-plugin"
### Optional
#### Directory
The directory that the plugin files will be created in. If undefined, it will create it
in the directory that it was executed in.
