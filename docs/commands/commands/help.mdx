# Help Command

This command lists every **default command** provided with the Scaped default CLI and their **basic functions**.
If more detailed explanations are needed, that is what these docs are for.

## Usage

```bash title="Command Terminal"
scaped help
```

## Arguements

This command takes in no arguements. You can stay safe knowing you just run it!
